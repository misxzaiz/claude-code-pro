src/components/UserList.tsx 复制 
import { useMemo } from 'react';

const UserList = ({ users, filter }) => {
  const filteredUsers = useMemo(() =>
    users.filter(u => u.name.includes(filter)),
    [users, filter]
  );

  return (
    <div>
      {filteredUsers.map(u => <User key={u.id} {...u} />)}
    </div>
  );
};

主要优化点：

  1. 使用 useMemo 缓存过滤结果
  2. 添加稳定的 key 属性
  3. 避免在渲染中创建新函数
