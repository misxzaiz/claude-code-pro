## 执行指南

### 开发新功能

1. 查看 `roadmap.md` 中优先级最高的未完成项
2. 确认实现方案和文件位置
3. 创建功能分支进行开发
4. 完成后更新进度表

### 代码优化

1. 查看 `optimization.md` 中的优化项
2. 优先处理高优先级（安全相关）项目
3. 每完成一项，在文档中标记完成状态
